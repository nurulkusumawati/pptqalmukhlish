# Pondok Tahfidzul Qur'an Al Mukhlish - Website

Situs statis sederhana yang berisi halaman terpisah: Beranda (`index.html`), Profil (`profil.html`), Berita (`berita.html`), Media Sosial (`medsos.html`), Daftar (`daftar.html`), Lokasi (`lokasi.html`), Brosur (`brosur.html`), Galeri (`galeri.html`), Kontak (`kontak.html`) dan Jadwal (`jadwal.html`).

Cara menggunakan:
- Salin folder ini ke `htdocs` (jika menggunakan XAMPP) atau buka `index.html` langsung di browser.
- Ganti teks dan gambar pada setiap file sesuai kebutuhan (gambar disimpan di `assets/images/` jika ingin mengganti file lokal).
- Untuk kustom lebih lanjut, edit `css/styles.css` dan `js/scripts.js`.

Tips:
- Ganti alamat email di formulir kontak dengan alamat yang valid.
- Tambahkan file brosur PDF ke `assets/images/` dan perbarui `brosur.html` jika ingin mengaktifkan unduh.
- Untuk membuat galeri lebih interaktif, pertimbangkan menambahkan Lightbox atau modal gambar dengan JavaScript.
